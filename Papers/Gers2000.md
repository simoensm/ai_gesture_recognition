## Abstract

The original [[Hochreiter1997]] LSTM did not have a forget gate — the cell state was only modified by additions. This paper introduces the **forget gate** $f_t = \sigma(W_f [h_{t-1}, x_t] + b_f)$, which allows the network to learn to reset its memory when context changes. This is now standard in all LSTM implementations.

---

## Key Contributions

- Introduced the **forget gate** to the LSTM architecture
- Demonstrated that the forget gate significantly improves performance on continual (unbounded) prediction tasks
- Modern PyTorch `nn.LSTM` includes the forget gate by default
