## Abstract

Comprehensive textbook on pattern recognition. Directly cited in the project guidelines as the reference for **edit distance** and **longest common subsequence (LCS)** — two of the three baseline methods required.

---

## Key Sections for This Project

- **Edit Distance (Levenshtein):** Chapter on string matching — provides formal DP derivation for the report
- **Longest Common Subsequence (LCS):** Chapter on sequence comparison — LCS is one of the three possible baselines (alongside DTW and edit distance)
- **k-NN Classification:** Foundation for the nearest-neighbor approach used in all three baselines
