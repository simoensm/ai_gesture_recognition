#### Minor points

- **Single dataset**: all conclusions are drawn from Huang et al. (2019). Caputo et al. (2018) benchmark on multiple datasets — your results might not generalise, which the conclusion section could note more explicitly.
- **Confusion matrices for one method only**: you analyse Three-Cents' confusion patterns in detail, but not the others'. A side-by-side comparison of where each method fails would strengthen the discussion (especially for DTW vs. Three-Cents, which share similar distance logic but very different performance).
- **BiLSTM: no variational dropout**: standard dropout applies a different mask at each timestep, which disrupts temporal gradient flow in LSTMs. Variational dropout (same mask across all tt t) is strictly better for recurrent networks (Gal & Ghahramani, 2016, cited in our Srivastava note). ## IMPROVEMENT 2 — DTW: Normalise accumulated cost by path length

Currently the raw D(N, M) is used as the distance. Replace it with:

    DTW_norm(X, Y) = D(N, M) / |W*|

where |W*| is the number of steps in the optimal warping path (i.e. the path 
length, not the sequence length). To get |W*|, backtrack through the DP table 
after computing D(N, M) to reconstruct the optimal path and count its steps.

This makes distances comparable across pairs of sequences with different lengths.
Apply this normalisation by default (no new hyperparameter needed).

## IMPROVEMENT 3 — Edit-distance: Normalise by sequence length

This bounds the distance in [0, 0.5] and removes the bias where longer sequences 
appear more dissimilar purely due to length. Apply this normalisation everywhere 
the edit distance is used as a k-NN metric.
